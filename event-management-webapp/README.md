# Event Horizon – Event Management Web App

This is a simple, responsive web app to view, add, search, and delete events. It uses plain HTML, CSS, and JavaScript.

## Features

- Predefined events rendered as cards
- Add new events (with validation)
- Delete events
- Automatic sort by date
- Highlight past events
- Search by name or date (YYYY-MM-DD)
- Responsive design (grid + flex)

## Files

- `index.html` – Page layout
- `style.css` – Styles and responsive rules
- `script.js` – Logic for rendering, sorting, search, add/delete
